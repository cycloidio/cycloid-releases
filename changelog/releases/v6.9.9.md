# [v6.9.9] _2026-04-15_

## Console

### **ADDED**
- **Plugins**: Plugins system [FE-1277](https://linear.app/cycloid/issue/FE-1277)
- **Logs**: Build log output now features an always-visible scrollbar, allowing you to quickly drag and navigate to any position in long outputs without relying on mouse-wheel scrolling [FE-1423](https://linear.app/cycloid/issue/FE-1423)

### **FIXED**
- **Dashboard**: Cloud cost management provider breakdown chart legend and tooltip display
- **Pipeline view**: URL copy button incorrectly redirecting to the anchor link [ENGFE-450](https://linear.app/cycloid/issue/ENGFE-450)
- **Roles**: Cloning a role from the roles list page now correctly pre-fills the new role form with the original role's name, description, and rules. Previously, clicking "Clone" would redirect to a blank create role page [ENGFE-451](https://linear.app/cycloid/issue/ENGFE-451)
- **Workers**: AWS region selector now correctly displays the region name (e.g. "US East (Ohio)") instead of showing a raw object value [ENGFE-452](https://linear.app/cycloid/issue/ENGFE-452)
- **Workers**: Scheduler fields (host, port, key) and region selection are no longer cleared when filling in the worker creation form in production [ENGFE-449](https://linear.app/cycloid/issue/ENGFE-449)

## API

### **ADDED**
- **Plugins**: Added the new Plugins feature [BE-1398](https://linear.app/cycloid/issue/BE-1398)
- **Other**: Added dlv to the development image along with a make debug-test command to debug tests in remote env.
